# parthchaudhari.com

Personal portfolio website for Dr. Parth Chaudhari.

**Live:** [https://parthchaudhari.com](https://parthchaudhari.com)

## About

Dr. Parth Chaudhari is an India-trained veterinary graduate, NAVLE-passed, currently working as a veterinary assistant/technician in Ontario, Canada, and founder of [DVMReady](https://dvmready.com).

## Tech Stack

- Static HTML5
- Custom CSS (design system with CSS custom properties)
- Vanilla JavaScript (mobile nav, smooth scroll, IntersectionObserver reveal animations)
- Hosted on GitHub Pages

## Structure

```
├── index.html          # Main portfolio page (all sections)

├── js/
```
